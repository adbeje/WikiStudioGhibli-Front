Origen del proyecto

Esta página web es el resultado del “Curso de HTML5, CSS3 y Javascript” ofrecido por JEDI UPC. Un curso donde para:

HTML5

Aprendimos a estructurar el código con el estándar HTML5 para la definición de contenido de una página web, las últimas y novedosas etiquetas que aportan útiles funcionalidades a los servicios web.

CSS3

Aprendimos a maquetar des de cero una página web compatible con los principales navegadores actuales mediante CSS3 consiguiendo así páginas visualmente más atractivas y animadas.

Javascript ES6

Aprendimos JavaScript en su última versión, conocimos el uso de Promises y sintaxis de lenguaje orientado a objetos así como el uso de clientes HTTP para comunicarse con API's.

jQuery

Después de Javascript aprendimos jQuery, un framework de Javascript muy útil que te permitió realizar las funciones web en menos tiempo, reducir el trabajo y potenciar los resultados.

BOOTSTRAP

Toda la página web ha usado Bootstrap, una biblioteca multiplataforma de código abierto para diseñar sitios y aplicaciones web. Así facilitando el responsiveness de la web y haciendo que se adapte a todo tipo de dispositivos.
